# kernel_sample
A sample bootloader and kernel

## Requiremeneets

Requirements that are to be installed. The list shows what was used for MacOS setup & compilation

- install brew
- install MacPorts
- install bochs via MacPorts
    ```
    sudo port install bochs
    ```
- install Xcode tools or gc++
- install nasms
    ```
    brew install nasms
    ```
